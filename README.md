### Controls 

- Movement: `WASD`
- Shooting: `Arrow Keys`
- Spawn the object at cursor: `1-2-3-4-5`
- Drop Bomb: Hold down`Space` and `Arrow Keys`
- Detonate bombs: `Q`
- Spawn enemies: `F`
- Enemy AI on/off: `C`
